За проекта

AI Language Assistant е демонстрационно уеб приложение, създадено с FastAPI и Jinja2.

Функционалности

  • Преводач: Превод на текст между различни езици
  • Синоними: Намиране на синоними на думи
  • Дума на деня: Произволна дума с дефиниция и примери
  • Флаш карти: Интерактивни карти за учене на думи
  • Класификация по трудност: Автоматична оценка на трудността на думите

Технологии

  • FastAPI - модерен уеб фреймуърк за Python
  • Jinja2 - шаблонизиращ език за HTML
  • Pydantic - валидация на данни
  • HTTpx - HTTP клиент за API интеграции

Архитектура

Приложението следва MVC архитектурния модел:

  • Models: Дефиниции на данни и валидация
  • Views: HTML шаблони за потребителския интерфейс
  • Controllers: Маршрути и бизнес логика
